My first day of web development

Aaron Lefkow

Dec 13 2022

Today marks my first day of learning web development. In middle school I took my first HTML class and enjoyed it but never fully pursued it as a professional path. I remember ny dad had even bought me a book on HTML and a license to the web development tool Adobe Dreamweaver. I've come back to web development many years later to build a specific marketable skill. Of course, part of the reason is also for personal enjoyment. There are many things I have wanted to create over the years (mostly video game related) but have not had the skills. This blog will help me to stay accountable in my web development journey. I have chosen to start my journey with the FrontendMasters Beginner Path. More specifically, I am starting with the Complete Intro to Web Development, v3 from Brian Holt. The main topics are below. Wish me luck!

  1. HTML
  2. CSS
  3. JavaScript
  4. The DOM
  5. Events and Listeners
  6. Talking to Servers
    • JSON
    • AJAX
    • async/wait
  7. Third Party Libraries
  8. Git and Github
